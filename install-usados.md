## dicas para instalar ou saber se já tem a biblioteca instalada
> * 1. verficar a versão do python instalada
```
python --version
```

> * 2. instalando
```
python -m pip install numpy
```

> * 3. Exemplo: python -m pip show nome_da_biblioteca
```
python -m pip show numpy
```

> * 4. listar todas bibliotecas instaladas
```
python -m pip list
```

> * 5. buscar por uma especifica
```
python -m pip list | findstr numpy
```
